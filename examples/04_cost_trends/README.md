# Cost Trends and Analytics

Track cost trends over time with moving averages and projections.

## What it shows

- Generate sample daily cost data
- Calculate moving averages
- Project monthly/annual costs
- Budget alerts

## Files

- `main.py` - Python example
- `run.sh` - Run the example

## Usage

```bash
./run.sh
```

## Key Takeaways

- Trend analysis for budget planning
- Moving averages smooth out daily fluctuations
- Projections help with annual budgeting
